# TODO app build with laravel 
<hr>


 ## usage :bulb: :

 - `git clone https://github.com/mustaphamajed/todo-app-back.git` 

- `cd todo-app-back`

- Edit `.env` and set your database connection details 

- `composer install`

- `php artisan migrate`  

- `php artisan serve --host YOUR-IP-ADRESS`
